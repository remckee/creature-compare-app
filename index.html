Mode info

In 2 creature mode, you can type in the name of a different living organism in each box to learn the most specific taxonomy category that includes both.

In 1 creature 1 feature mode, you can type in the name of an organism in the first box and any term or phrase in the second box to learn any facys about the organism that mention the feature in the second box.

Note that, despite the use of the term "creature", the organism can be in any kingdom, including Animal, Plant, Fungus, Protist, or Bacteria.
